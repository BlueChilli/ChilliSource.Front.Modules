# responsive menu


These dependencies are required

```
yarn add sreact-burger-menu react-responsive
````

Usage:

```
<ResponsiveMenu>
  <ul>
    <li><Link to="/account/signin">Sign In</Link></li>
    <li><Link to="/account/signout">Sign Out</Link></li>
  </ul>
</ResponsiveMenu>

```


CSS required:

*Hamburger (mobile) mode*



*Desktop mode*
